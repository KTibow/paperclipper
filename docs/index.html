How a paperclip is tested

Every unit is a continuous wire centreline checked in two independent code paths (closed-form & sampled). It only ships if it passes all of:

Well-formed wire

Four ordered, finite lanes forming a single continuous wire with two free ends.

Manufacturable size

Footprint and wire length fall within real paperclip tolerances.

Clamping function

At least two overlapping channels with a gap that actually captures & grips paper.

No fusion

Adjacent wire edges keep clearance, so the wire never touches or crosses itself.

Ends tucked

Both free ends sit safely in the top half so the clip won't snag.

Reproducible

Every batch carries a deterministic checksum; the whole ledger is independently re-verified.
